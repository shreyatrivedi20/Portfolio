# 🌐 Shreya Trivedi — Personal Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

🔗 **Live Site:** [portfolio-henna-two-33.vercel.app](https://portfolio-henna-two-33.vercel.app)

---

## ✨ Features

- **Scroll Animations** — smooth reveal effects on all sections as you scroll
- **Dark / Light Mode Toggle** — switch themes with one click
- **Certifications Section** — showcasing verified achievements
- **Responsive Design** — works on mobile, tablet, and desktop
- **Gradient Headings** — animated color transitions on all section titles
- **Contact Form** — built-in message form with submit feedback

---

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, animations, flexbox, grid)
- Vanilla JavaScript (IntersectionObserver, theme toggle)
- Google Fonts — Space Grotesk & Inter
- Deployed on **Vercel**
